# 游戏房间满员状态监测工具 v2.0

## 功能特性
- 支持同时监控最多3个游戏窗口
- 后台监控技术，窗口被遮挡也能正常工作
- 交互式区域选择，精确框选监测图标
- 智能颜色检测，识别满员（绿色）和空缺状态
- 自定义提示音或使用系统蜂鸣声
- 实时日志记录（24小时制）

## 安装依赖
```bash
pip install -r requirements.txt
```

## 使用方法
```bash
python game_monitor.py
```

### 操作流程
1. **添加任务**：点击"➕ 添加窗口位置"
2. **选择窗口**：切换到游戏窗口使其置顶，点击确定
3. **框选区域**：在半透明遮罩层上拖拽框选满员图标区域
4. **配置任务**：
   - 双击任务行切换 ON/OFF
   - 右键菜单：修改备注、导入提示音、删除任务
5. **启动监控**：点击"▶ 总启动"

## 核心技术
- **后台捕获**：使用 `BitBlt` 获取窗口DC，支持被遮挡窗口
- **颜色检测**：绿色判定逻辑 `G > 130 且 G > R + 40`
- **状态触发**：仅在"满员→空缺"切换时播放提示音

## 系统要求
- Windows 10 / Windows 11
- Python 3.7+
- 建议管理员权限运行
